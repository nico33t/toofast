Audit a site's performance using the nt-deploy MCP tools.

1. Ask for a URL or client name (and mobile/desktop, default mobile).
2. Call the `nt_audit` tool.
3. Summarize the scores (Performance, Accessibility, Best Practices, SEO) and Core Web Vitals,
   then suggest concrete fixes for anything below 90.
